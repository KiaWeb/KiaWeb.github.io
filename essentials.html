Getting Started

WARNING: The asset may seem private due to the permissions being in beta.

To use the Dev Essentials, insert the code into your script, and use the API to begin scripting!

local Essentials = require(84466135090536) -- Module code!

Once required, you can call Essentials functions like :Notify() or :CreateSpriteClip().

API Reference

Essentials:EnableAE()

  • Description - Enables AntiExploit module (Cannot be disabled unless the function is never called)

Essentials:SetAEConfig()

  • Description - Sets the AntiExploit configuration values.
  • Parameters - table containing the following keys:
    • EnforceWalkSpeed (boolean) - Enforce server authority on walkspeed.
    • AllowedWalkSpeed (number) - Default walkspeed (e.g., 16).
    • WalkSpeedLeeway (number) - Wiggle room before resetting/striking.
    • EnforceJumpPower (boolean) - Enforce server authority on jump power.
    • AllowedJumpPower (number) - Default jump power (R15 commonly 50).
    • JumpPowerLeeway (number) - Wiggle room before resetting/striking.
    • SpeedSpikeEnable (boolean) - Enable anti-teleport/speed spike detection.
    • MaxRunSpeed (number) - Max legit ground speed (sprinting, buffs).
    • MaxAirSpeed (number) - Max legit air speed (knockback cases).
    • TeleportDistance (number) - One-frame "blink" distance suspicious.
    • GraceOnRespawnSeconds (number) - Grace period after respawn in seconds.
    • NoclipCheckEnable (boolean) - Enable noclip detection.
    • NoclipSampleInterval (number) - Interval in seconds between checks.
    • MaxSolidContacts (number) - Number of solid parts intersecting HRP bounds before suspecting noclip.
    • RemoteSpamWindow (number) - Time window in seconds for remote spam detection.
    • RemoteSpamMaxCalls (number) - Max calls per window per remote before flagging.
    • StrikeToWarn (number) - Number of strikes before warning.
    • StrikeToKick (number) - Number of strikes before kicking the player.
    • StrikeToTempBan (number) - Number of strikes before temporary ban (if implemented).
    • Debug (boolean) - Enable or disable debug messages.
  • Example not available.

Essentials:GetVersion()

  • Description - Returns a (string) value containing the version of the Dev Essentials

Essentials:Notify(data,player)

  • Title (string) - Notification title
  • Text (string) - Notification message
  • Image (string) - Roblox asset ID for icon
  • Duration (number) - Time in seconds

Essentials:CreateSpriteClip(sheetData)

  • Image (string) - Spritesheet asset ID
  • FrameSize (Vector2) - Frame size in pixels
  • Frames (number) - Total frames
  • Columns (number) - Frames per row
  • FPS (number) - Animation speed
  • Loop (bool) - Should it loop

Essentials:WhitelistPlayers(players)

  • Description (table) - Table with all Player instances, whitelists them from the AntiExploit module

Examples

-- Simple notification
Essentials:Notify({
    Title = "Welcome!",
    Text = "Thanks for playing!",
    Image = "rbxassetid://123456789",
    Duration = 5
},game.Players:GetPlayerFromCharacter("myUsernameOrWhatever"))
print(Essentials:GetVersion())
Essentials:EnableAE()
Essentials:WhitelistPlayers({game.Players:GetPlayerFromCharacter("myUsernameOrWhatever")})
-- Spritesheet animation
local myImageLabel = game.StarterGui.MyImageLabel
local mySpriteClip = Essentials:CreateSpriteClip(myImageLabel, {
    Image = "rbxassetid://987654321",
    FrameSize = Vector2.new(64,64),
    Frames = 16,
    Columns = 4,
    FPS = 12,
    Loop = true
})
mySpriteClip:Play(myImageLabel)
-- or
mySpriteClip:Stop()
